# MVP 规则表（大地图势力经营版）

## 1. 全局规则

| 模块 | 规则说明 | 建议参数 | 备注 |
|---|---|---|---|
| 地图结构 | 多势力共存的大地图 | 25-40 座城市 | 首版用 25 城减负 |
| 势力数量 | 玩家 + AI 势力 | 4-6 个势力 | 先做 4 势力最稳 |
| 回合单位 | 月份回合制 | 1 回合 = 1 个月 | 便于资源结算 |
| 胜利条件 | 统一全地图城市 | 全城占领 | 支持长期目标 |
| 失败条件 | 玩家势力灭亡 | 无城池或主城失守 | 明确终局 |

## 2. 回合流程规则

| 阶段 | 内容 | 关键产出 |
|---|---|---|
| 内政阶段 | 城池生产、建筑施工、治安管理 | 粮草、金钱、兵员、民心变化 |
| 人才阶段 | 任命官员、招募/登用、忠诚波动 | 人才池与城池效率变化 |
| 军事阶段 | 征兵、编队、调兵、出征 | 部队位置与战线变化 |
| 外交阶段 | 结盟、停战、交易、离间 | 势力关系变化 |
| 结算阶段 | 收益、消耗、事件触发、胜负判定 | 进入下回合 |

## 3. 城池与内政规则

### 3.1 城池基础属性
| 属性 | 说明 | 建议范围 |
|---|---|---|
| 人口 | 影响兵员与税收上限 | 1,000-20,000 |
| 粮草 | 军队维持与出征消耗 | 0-999,999 |
| 金钱 | 建设、招募、外交消耗 | 0-999,999 |
| 治安 | 影响叛乱与产出效率 | 0-100 |
| 民心 | 影响长期稳定与增长 | 0-100 |

### 3.2 内政动作（MVP）
| 动作 | 效果 | 成本 |
|---|---|---|
| 农业开发 | 增加粮草月产 | 消耗金钱 + 行动力 |
| 商业开发 | 增加金钱月产 | 消耗粮草 + 行动力 |
| 治安整顿 | 降低叛乱概率 | 消耗行动次数 |
| 训练驻军 | 提升部队士气/战斗力 | 消耗粮草 |

## 4. 军事与战斗规则

### 4.1 出征与行军
| 规则项 | 建议 |
|---|---|
| 出征条件 | 城池有兵力、粮草、武将 |
| 行军消耗 | 每移动 1 格消耗粮草 |
| 补给规则 | 脱离己方城市补给线会降士气 |
| 攻城条件 | 敌城守军低于阈值或城防被破 |

### 4.2 战场战斗（网格）
| 模块 | 规则说明 | 建议参数 |
|---|---|---|
| 地图 | 战斗地图网格 | 16x16 或 20x20 |
| 行动 | 每单位每回合 1 次行动 | 移动后可攻击 |
| 伤害 | 攻防差值结算 | 伤害 = 攻击 - 防御（最低 1） |
| 命中 | 命中率判定 | 命中 = 基础命中 - 闪避 + 地形修正 |
| 兵种克制 | 以 `需求完善_闭环检查_v1.md` 冻结口径为准（骑兵->戟盾兵->枪兵->骑兵，弓兵/投石车按特殊规则） | 倍率参数按冻结表执行 |
| 建筑陷阱 | 城防建筑和陷阱触发伤害/减速 | 每城 1-2 种 |

## 5. 技能与计策规则

| 规则项 | 建议 |
|---|---|
| 技能数量 | 每名核心武将 1 主动技能 |
| 计策类型 | 火计、扰乱、鼓舞、伏兵（首版选 2-3 个） |
| 冷却机制 | 2-4 回合 CD |
| 触发代价 | 消耗计策点（由智力人才提供） |
| 表现成本 | 共用特效模板，先逻辑后演出 |

## 6. 外交规则（MVP）

| 外交动作 | 结果 | 成本/风险 |
|---|---|---|
| 提议停战 | 指定回合内互不进攻 | 关系要求较高 |
| 结盟 | 双方关系提升、减少被多线夹击 | 需持续维护 |
| 资源交易 | 粮草与金钱互换 | 价格受关系影响 |
| 离间（计策） | 降低目标势力关系/忠诚 | 失败会反噬关系 |

## 7. 人才规则（MVP）

| 模块 | 规则 |
|---|---|
| 基础属性 | 统率、武力、智力、政治、魅力 |
| 任命岗位 | 太守、军师、都督（3 类岗位） |
| 忠诚系统 | 低忠诚存在被挖角风险 |
| 登用方式 | 搜索、劝降、俘虏招降 |
| 成长方式 | 随回合微成长或战功成长 |

## 8. AI 势力规则（MVP）

1. AI 每回合优先保证主城治安和粮草安全线。
2. 若邻近玩家且军力占优，优先发动单线进攻。
3. 劣势时优先求和或结盟，避免灭亡过快。
4. AI 以“占城优先”而非“歼灭优先”进行战略选择。

## 9. 数值基线（首版建议）

| 系统 | 参数建议 |
|---|---|
| 回合总时长 | 一局 120-240 回合（20 小时+） |
| 初始持城数 | 玩家 2 城，AI 各 3-6 城 |
| 每城月收入 | 粮草 300-1500，金钱 200-1000 |
| 出征基础消耗 | 每部队每回合粮草 20-60 |
| 叛乱阈值 | 治安 < 40 时概率触发 |

## 10. 暂不纳入 MVP
- 海战、水军与港口专属系统。
- 复杂科技树与时代演进。
- 历史事件链与大型剧情演出。
- 多层级官职和家族系统。
