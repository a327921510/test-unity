# MVP 规则表（索引版）

## 1. 文档定位

本文件仅用于定义 MVP 范围与验收口径，不再承载具体规则细则与数值公式。  
具体规则以模块设计文档为唯一真源。

## 2. MVP 范围边界

| 类别 | MVP 纳入 | 暂不纳入 |
|---|---|---|
| 核心循环 | 经营生产 -> 编制出征 -> 战斗攻城 -> 占领扩张 | 海战/港口专属系统 |
| 地图系统 | 大地图势力轮转、城市占格与机动规则 | 复杂多层地图机制 |
| 军事系统 | 部队编制、出征、补给、攻城、战损 | 复杂兵科科技树 |
| 城市系统 | 人口/治安、产出、基础建设、耐久 | 深度剧情型城市事件链 |
| 规则治理 | 关键阈值与输入限制可配置 | 多版本并行平衡分支 |

## 3. 回合与胜负口径（验收级）

- 回合、行动、胜负等具体规则不在本文件维护，统一引用模块文档与模式配置。

## 5. MVP 验收清单（最小）

- 经营侧：城市月结算可产出可消费资源，并影响可编制能力。
- 军事侧：可完成编制、出征、移动、战斗、入城拆解、补给闭环。
- 地图侧：地图与空间规则按模块文档生效并通过基础可玩验证。
- 后勤侧：运输兵补给与断粮惩罚生效，资源上限截断口径一致。
- 战果侧：攻城占领可改变城市归属并回流经营循环。
