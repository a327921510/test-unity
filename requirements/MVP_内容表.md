# MVP 内容表（清单版）

## 1. 文档定位

本文件只保留“内容资产与制作清单”，不维护规则公式与结算逻辑。  
规则细节统一引用模块文档与 `需求完善_闭环检查_v1.md`。

## 2. 势力与地图内容清单

| 清单项 | 建议规模 | 说明 |
|---|---:|---|
| 势力数量 | 4-6 | MVP 建议先做 4 |
| 每势力初始城市 | 2-6 | 用于形成早中期差异 |
| 城市总数 | 25-40 | 首版建议 25 降低负担 |
| 城市类型 | 主城/农业城/商业城/军事城/关隘城 | 只定义内容标签，不在此定义规则 |

## 3. 城市与建筑内容清单

| 类型 | 最小清单 |
|---|---|
| 城市面板内容 | 粮草、金钱、兵源、治安、战乱状态、耐久 |
| 建筑类型 | 农田、市场、兵营、城防工事、陷阱工坊、外交馆 |
| 产出表现 | 月度结算条目与增减来源可追溯 |

## 4. 部队与兵种内容清单

| 类型 | 最小清单 |
|---|---|
| 兵种 | 枪兵、骑兵、弓兵、戟盾兵、投石车、运输兵 |
| 部队展示属性 | 兵力、基础攻防、移动、状态、资源携带 |
| 战场表现 | 行动范围、攻击提示、受击反馈、溃败反馈 |

## 5. 人才与策略内容清单

| 类型 | 最小清单 |
|---|---|
| 人才分类 | 军事将领、内政官、谋士、通用人才 |
| 人才面板 | 核心属性、忠诚、任命状态 |
| 策略内容 | 基础增益/控制/伤害类策略（MVP 2-3 个） |

## 6. 外交与事件内容清单

| 类型 | 最小清单 |
|---|---|
| 外交动作 | 停战、结盟、贸易、离间（MVP 可裁剪） |
| 事件类型 | 丰收、叛乱（MVP）；其他事件后续扩展 |
| 日志要求 | 外交结果与事件影响均可回放 |

## 7. UI 与资源制作清单

### 7.1 UI 最小清单
- 大地图总览：势力边界、城池归属、前线状态。
- 城市面板：资源、治安、耐久、可编制能力。
- 战场面板：行动范围、结算提示、地形与状态信息。
- 月度结算：收支变化、战果、关键事件。

### 7.2 美术资源最小清单
- 世界层：地块（平原/山地/森林/水域/道路/城池）与势力配色。
- 战场层：兵种图块、城防与陷阱图标、状态图标。
- UI 层：资源图标、关系图标、通用按钮与面板皮肤。

## 8. 制作优先级（从高到低）

1. 大地图回合与城市资源结算闭环。
2. 编制、出征、战斗、补给、入城拆解闭环。
3. 人才任命与基础外交流程。
4. 策略与陷阱接入。
5. 事件系统与表现打磨。
